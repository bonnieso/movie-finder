## Instructions

In order to run the application, you must follow the instructions below for and run both server and client.

### Running the server

In your terminal, run the following commands:

`cd server`

`npm install`

`APIKEY='your_api_key' npm start`

Replace your_api_key with your api key from [The Movie Database API](https://developers.themoviedb.org/3/getting-started/introduction).

### Running the client:

In a separate terminal window, run the following commands:

`cd client`

`npm install`

`npm start`

Open [http://localhost:3000](http://localhost:3000) to view the app in the browser.

### Navigating the app:

Type in a movie name in the search box and hit enter.

Click on a movie poster to view details.

Click "back" to go back to search results.

To view popular movies again, just hit enter in the empty search box.
